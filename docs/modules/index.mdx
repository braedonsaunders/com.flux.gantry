## The Nine Dashboards

Gantry provides nine specialized dashboards, each focused on a critical area of financial analysis. Together, they give you complete visibility into your organization's financial health.

## Dashboard Summary

| Dashboard | Purpose | Key Question It Answers |
|-----------|---------|------------------------|
| [AI Advisor](/docs/gantry/modules/ai-advisor) | Natural language queries | "What do I need to know right now?" |
| [Liquidity](/docs/gantry/modules/liquidity) | Cash flow analysis | "How long can we operate?" |
| [Profitability](/docs/gantry/modules/profitability) | Margin analysis | "Are we making money?" |
| [True Cost](/docs/gantry/modules/true-cost) | Overhead rates | "What do things really cost?" |
| [Billable IQ](/docs/gantry/modules/billable-iq) | Utilization tracking | "Are our people productive?" |
| [Sentinel](/docs/gantry/modules/sentinel) | Fraud detection | "Is anything suspicious happening?" |
| [Procurement](/docs/gantry/modules/procurement) | Vendor performance | "Are our vendors performing?" |
| [Revenue Intelligence](/docs/gantry/modules/revenue-intelligence) | Customer health | "Which customers need attention?" |
| [Spend Velocity](/docs/gantry/modules/spend-velocity) | Spending patterns | "Where is money going?" |

## Choosing Where to Start

Different roles will find different dashboards most valuable:

### For CFOs and Controllers
Start with **Profitability** for overall health, then **Liquidity** for cash position. Use **Sentinel** to ensure transaction integrity.

### For AP/AR Managers
Focus on **Liquidity** for cash flow timing, **Procurement** for vendor management, and **Revenue Intelligence** for customer payment patterns.

### For Department Heads
Check **Billable IQ** for team utilization and **True Cost** to understand your department's overhead contribution.

### For Controllers and Auditors
**Sentinel** provides fraud detection and anomaly analysis. **Profitability** shows margin trends.

### For Everyone
The **AI Advisor** lets you ask questions in plain English across all areas.

## Dashboard Categories

### Financial Health
- **Profitability** — Overall financial performance and margin analysis
- **Liquidity** — Cash position and runway forecasting

### Operational Efficiency
- **True Cost** — Overhead rates and cost recovery
- **Billable IQ** — Employee utilization and productivity

### Risk & Compliance
- **Sentinel** — Fraud detection and transaction integrity

### Stakeholder Analytics
- **Procurement** — Vendor performance and spend analysis
- **Revenue Intelligence** — Customer health and segmentation

### Pattern Detection
- **Spend Velocity** — Spending trend analysis and anomaly detection

### Intelligence Layer
- **AI Advisor** — Natural language interface to all dashboards

## Common Features

All dashboards share these capabilities:

### Date Range Selection
Choose the time period to analyze. Options include:
- Last 30/60/90 days
- This fiscal year
- Last fiscal year
- Custom date range

### Refresh Button
Reload data from NetSuite without refreshing the entire page.

### Export Options
Where available, export data to CSV or Excel for further analysis.

### Color-Coded Indicators
Consistent color scheme across all dashboards:
- **Green** — Healthy/Good
- **Yellow/Amber** — Warning/Needs attention
- **Red** — Critical/Urgent action needed
- **Gray** — Neutral/Informational

### Drill-Down Capability
Click on metrics, chart elements, or table rows to see underlying details.

### Tooltips
Hover over any metric or chart element for explanations and additional context.

## Dashboard Requirements

Some dashboards require specific NetSuite features:

| Dashboard | Required Feature |
|-----------|-----------------|
| Billable IQ | Time Tracking |
| Procurement | Purchase Orders |
| True Cost | Projects (optional, enhances analysis) |
| Multi-Subsidiary analysis | OneWorld |

Dashboards work without optional features but provide richer analysis when enabled.

## Configuring Dashboards

Each dashboard has configurable thresholds and settings. Access them through:

1. **Settings page** — Central configuration for all dashboards
2. **Dashboard options** — Some dashboards have inline settings

See [Configuration](/docs/gantry/configuration) for detailed setup instructions.
