## License Overview

Gantry requires a valid license to operate. Your license determines which features and dashboards you can access.

## License Tiers

| Tier | What's Included |
|------|-----------------|
| **Starter** | Basic dashboard access for small teams |
| **Standard** | All core dashboards and features |
| **Professional** | Advanced features including AI Advisor |
| **Enterprise** | Full feature set with API access and premium support |

Contact your Flux sales representative for detailed tier comparison and pricing.

## Activating Your License

### First-Time Activation

1. Open Gantry after installation
2. Click **Settings** in the sidebar
3. Go to the **License** tab
4. Enter your license key in the field provided
5. Click **Verify License**

If successful, you'll see:
- License status: **Active**
- Licensed organization name
- Tier level and badge
- Expiration date
- Activated modules

### License Key Format

Your license key is a unique alphanumeric code provided by Flux. It typically looks like:

```
GNTRY-XXXX-XXXX-XXXX-XXXX
```

> ⚠️ **Warning**: Keep your license key confidential. It's tied to your organization and should not be shared publicly.

## License Validation

Gantry validates your license automatically:

- **On startup** — Checks license when you open Gantry
- **Periodically** — Revalidates every 24 hours
- **On demand** — When you click Verify License in Settings

### Offline Mode

If Gantry can't reach the license server (network issues):

- **24-hour grace period** — Gantry continues working normally
- **After grace period** — Features may be limited until connectivity is restored

This ensures temporary network issues don't interrupt your work.

## Managing Your License

### Viewing License Details

In **Settings → License**, you can view:

| Field | Description |
|-------|-------------|
| **Status** | Active, Expired, or Invalid |
| **Licensed To** | Your organization name |
| **Tier** | Your current license tier |
| **Expiration** | When your license needs renewal |
| **Activated** | When the license was first activated |
| **Modules** | Features enabled for your tier |

### Upgrading Your License

To upgrade to a higher tier:

1. Contact sales@fluxfornetsuite.com
2. You'll receive a new license key
3. Enter the new key in Settings → License
4. New features are available immediately

### Renewing Your License

You'll receive renewal notices before your license expires. To renew:

1. Contact your Flux representative or visit fluxfornetsuite.com
2. Complete the renewal process
3. If your key changes, enter the new key in Settings

If your key remains the same, Gantry automatically recognizes the renewal.

## Sandbox vs Production

Gantry licenses are environment-specific:

- **Production license** — Works only in your production NetSuite account
- **Sandbox license** — Works only in sandbox/test accounts

If you need both, contact Flux for a sandbox license. Attempting to use a production license in a sandbox (or vice versa) will show an invalid license error.

## License Issues

### "Invalid License" Error

This appears when:
- The license key is entered incorrectly
- The license has expired
- The license is for a different environment (production vs sandbox)
- The license has been revoked

**Solution**: Double-check your key, verify the environment, and contact support if issues persist.

### "License Expired" Error

Your license term has ended.

**Solution**: Contact sales@fluxfornetsuite.com to renew.

### "Unable to Validate" Error

Gantry can't reach the license server.

**Solution**:
1. Check your internet connection
2. Verify firewall settings allow outbound HTTPS connections
3. Wait for the grace period (24 hours) if the issue is temporary
4. Contact support if the problem persists

### "Wrong Environment" Error

You're using a production license in a sandbox or vice versa.

**Solution**: Use the correct license for your environment, or contact Flux for an additional license.
